# Bienvenu(e) sur la documentation !

Bienvenu(e) sur la documentation interne de Developer's House.
Cette documentation détaille la structure interne et les services
dédiés à developer's House.

### A propos de la confidentialité de la documentation

Cette documentation est censée demeurer privée. Mais si vous
devez partager quelconque partie de cette documentation, vous
devez demander à un administrateur Developer's House avant de la
partager.

> Ne partagez pas cette documentation sans l'autorisation d'un
administrateur de Developer's House.

### Structure

La documentation est structurée en plusieurs parties représentant
les API de base de Developer's House

* Services d'authentification
* Services de comptes
* Infrastructure

et beaucoup d'informations utiles pour les développeurs comme la
configuration des informations d'identification, le monitoring,
les tutoriels et plus encore.
