# Entry Tests

Technical interview questions when recruiting someone. Inspired by [Google Foobar](https://foobar.withgoogle.com).

Questions must be kept secrets!!

## Usage

To run an interview using questions in this repository, please refer to the [docs/INTERVIEW.md](./docs/INTERVIEW.md) file.

To add a question, please refer to the [docs/ADD_QUESTION.md](./docs/ADD_QUESTION.md) file.
