# ~ Cryir - The centralized data controller.

Cryir is the one of the core components of the data on the whole developer's house architecture.
It handles the requests of data takeout, data deletion and account management in every service of developer's house.
Is's the component that handles the [dataanager](https://gitlab.com/developers-house/modules/datamanager) library.

This component is closed source and always will be as it's a custom component of our infrastructure.
