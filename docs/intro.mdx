# Welcome to the documentation!

Welcome to the internal documentation of Developer's House.
This documentation details the internal infrastructure and services
dedicated to Developer's House. 

### A word about the sharing of this documentation

This documentation is supposed to be private. But if you 
need to share any part of this documentation containing 
internal systems's documentation, you can ask a maintainer
of the infrastructure for the permission.

> Do not share this documentation without the authorization of a project maintainer.

### Structure

The documentation is separated in multiple parts representing
the core apis of Developer's House

* Login services
* Account services
* Infrastructure

and many Developer-Related stuff such as credentials setup,
monitoring tutorials and more.
