# Add a question

Questions consist of an algorithm problem with a set of test cases.

## Criteria

To find a question, you must follow some guidelines to ensure the problem is a good fit for anyone.

### Time limit

While the time limit is not a strict requirement, we recommend that a problem should not take more than 48 hours to complete.

### Difficulty

A question should not be too complex. We use Google *Foobar*'s level 2 challenges as a references.

### Language agnostic

It is true that some programming languages are easier to use than other and or are more appropriate for certain problems. 
However, questions in this repository are algorithm questions, not programming language questions. This means that issues should be resolvable in any programming language.

Nevertheless, since we cannot support every programming language on the infrastructure, answers must be written using one of the following languages:
- Python (3.6+)
- JavaScript / TypeScript (Node.js v10+)

This means that you must commit tests in all of these languages!

## Committing

When you are ready with the problem and want to add it to the repository, you must create a new folder in the `questions/` directory.

You can duplicate `questions/__template__/` and rename it following the **snake_case** pattern.

In the directory, you will find a myriads of files, which are important to keep the repository organized.
